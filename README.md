# Kawanusa

Kawanusa hadir untuk mengajak anak-anak Indonesia mengenal, mencintai, dan melestarikan budaya Nusantara melalui pengalaman belajar yang interaktif dan menyenangkan. Kami ingin mengajak setiap anak menjelajahi Nusantara, memahami keberagaman, serta menumbuhkan rasa bangga untuk menjaga warisan budaya Indonesia bagi generasi masa depan.

## Fitur

- Landing Page
- Jelajah Budaya
- Detail Provinsi
- Mini Games
- Cerita Nusantara
- Detail Cerita

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Framer Motion

## Struktur Folder

- `src/assets`: Berisi seluruh aset statis seperti gambar (Hero, Ikon, Budaya) dan fonts.
- `src/components`: Berisi komponen UI React yang dapat digunakan ulang (Reusable Components) seperti Navbar, Footer, Card, dan elemen interaktif lainnya.
- `src/data`: Berisi file data statis seperti soal kuis, data provinsi, gambar tebakan, dan JSON konten budaya.
- `src/hooks`: Berisi custom React Hooks untuk memisahkan logic (seperti useGameEngine, useQuiz) dari tampilan UI.
- `src/pages`: Berisi komponen utama untuk setiap halaman (Landing Page, Jelajah Budaya, Mini Games, Cerita, dll).
- `src/router`: Berisi konfigurasi routing aplikasi menggunakan React Router.
- `src/utils`: Berisi fungsi utilitas dan helper (misalnya animasi transisi Framer Motion).
